# Korean PDF Writer

한국어 원문이나 마크다운 초안을 단정한 인쇄용 문서 미리보기로 정리하고, 브라우저의 인쇄 기능으로 PDF를 저장하는 웹 도구입니다.

## 특징

- 한국어 장문에 맞춘 세리프 기반 문서 미리보기
- 제목, 문단, 목록을 단순 마크다운 문법으로 변환
- 양피지 배경과 잉크블루 단일 강조를 사용하는 절제된 화면
- 별도 서버 저장 없이 브라우저 안에서 작성·미리보기·인쇄

## 실행

```bash
npm run dev
```

브라우저에서 `http://localhost:3000`을 열고 문서를 붙여 넣은 뒤 **PDF로 저장**을 누르면 인쇄 대화상자가 열립니다.

## 검증

```bash
npm run build
```

별도 런타임 의존성 없이 정적 HTML, CSS, JavaScript로 동작합니다.
