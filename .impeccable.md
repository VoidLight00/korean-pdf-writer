# Impeccable Design Context

## Design Context

### Users
한국어 문서를 PDF로 정리해야 하는 강사, 컨설턴트, 연구자, 개발자, 1인 지식 생산자. 사용자는 초안 마크다운이나 원문 메모를 붙여 넣고, 문서 목적·독자·톤에 맞는 단정한 PDF와 보조 SVG 다이어그램을 빠르게 만들고 싶어 한다.

### Brand Personality
조용함, 정밀함, 중립성. 인터페이스는 특정 개인 브랜드가 아니라 범용 오픈소스 문서 도구처럼 느껴져야 하며, 사용자가 정한 문서 맥락을 앞세워야 한다.

### Aesthetic Direction
양피지 배경, 잉크블루 단일 강조, 세리프 중심의 고급 문서 편집 도구. 밝은 테마를 기본으로 하며 그라데이션, 네온, 과한 카드, 이모지, 로고 워터마크, AI SaaS식 영웅 섹션을 피한다.

### Design Principles
1. 한국어 장문 가독성을 최우선으로 한다.
2. 사용자가 답한 문서 유형, 독자, 톤을 미리보기 상단에 반영한다.
3. SVG는 본문 의미를 보조하는 단순 구조물로만 삽입한다.
4. 모든 상호작용은 문서 작성 흐름을 방해하지 않아야 한다.
5. 개인 로고나 특정 브랜드에 의존하지 않는 범용 오픈소스 도구로 유지한다.
