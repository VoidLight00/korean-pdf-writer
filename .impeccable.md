# Impeccable Design Context

## Design Context

### Users
한국어 문서를 PDF로 정리해야 하는 강사, 컨설턴트, 연구자, 1인 지식 생산자. 사용자는 초안 마크다운이나 원문 메모를 붙여 넣고, 인쇄·공유 가능한 단정한 PDF 문서로 빠르게 다듬고 싶어 한다.

### Brand Personality
조용함, 정밀함, 신뢰감. 인터페이스는 사용자를 설득하려 들지 않고, 문서가 스스로 품위를 갖추도록 뒤에서 돕는 편집실처럼 느껴져야 한다.

### Aesthetic Direction
양피지 배경, 잉크블루 단일 강조, 세리프 중심의 고급 문서 편집 도구. 밝은 테마를 기본으로 하며 그라데이션, 네온, 과한 카드, 이모지, 둥근 아이콘 그리드, AI SaaS식 영웅 섹션을 피한다.

### Design Principles
1. 한국어 장문 가독성을 최우선으로 한다.
2. 장식보다 여백, 행간, 대비, 종이 질감으로 품질을 만든다.
3. 모든 상호작용은 문서 작성 흐름을 방해하지 않아야 한다.
4. 브랜드 색은 잉크블루 하나만 절제해서 사용한다.
5. 공개 제품 문구는 특정 기반 프로젝트가 아니라 한국어 PDF 문서 작성 경험을 중심으로 쓴다.
